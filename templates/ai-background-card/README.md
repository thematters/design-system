# AI Background Card

Square campaign graphic for generated backgrounds plus deterministic Matters
layout.

## Size

`1080 x 1080`

## Data

| Key             | Required | Notes                                                                                                                      |
| --------------- | -------- | -------------------------------------------------------------------------------------------------------------------------- |
| `backgroundUrl` | yes      | Local path, file URL, or HTTPS URL. Generated by `scripts/create-campaign-visual.mjs` when `background.mode = "generate"`. |
| `kicker`        | yes      | Short campaign label.                                                                                                      |
| `headline`      | yes      | Main Chinese headline. Clamped for square layout.                                                                          |
| `deck`          | yes      | One-sentence supporting copy.                                                                                              |
| `cta`           | yes      | Footer domain or action.                                                                                                   |

## Render

```bash
pnpm visual:create -- --job brand/jobs/background-card.example.json
```

With `OPENAI_API_KEY` and `background.mode = "generate"`, the script creates a
new background first, then renders the final PNG.
