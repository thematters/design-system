# Components

本資料夾將鏡射 Figma `Product - Design system` 內的 Components。目前為空。

## 導入順序建議

1. 原子級（Button、Input、Tag、Avatar、Icon）
2. 複合級（Card、Toast、Dialog、Menu）
3. 場景級（ArticleCard、UserCard、CircleCard、NotificationItem）

每個 component 建議一個子資料夾，包含：
- `*.tsx`（或 `.vue` / `.astro`，依前端選型）
- `*.stories.tsx`
- `*.test.tsx`
- `README.md`（用途、props、Figma 對應節點）
