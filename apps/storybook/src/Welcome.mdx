# Matters Design System

Storybook docs for [`thematters/design-system`](https://github.com/thematters/design-system).

## What's in here

- **Tokens** — Brand · Colors · Typography · Spacing · Shadows · Freewrite (opt-in)
- **Components** — Button · TextField · Dialog · Toast (Phase 1–2)
- More components arrive in Phase 4 (ArticleCard / Avatar / Banner)

## Three consumption channels

| Channel        | Best for                                  | Entry                                                 |
| -------------- | ----------------------------------------- | ----------------------------------------------------- |
| Vendored copy  | Static landing pages / no JS framework    | `cp -r tokens/dist consumer/assets/`                  |
| **npm package** | React apps                                | `pnpm add @matters/design-system-react`               |
| Copy-and-own   | Forks / one-off marketing sites           | `node scripts/scaffold-component.mjs Button`          |

Full consumer guide: [`docs/consume.md`](https://github.com/thematters/design-system/blob/main/docs/consume.md).
