# Matters Design System — Storybook

This is the local docs site for `@matters/design-system-react`.

**Status:** Phase 1 — only `Button` is shipped here. Phase 2 will add
`TextField`, `Dialog`, `Toast`. Phase 3 will deploy this site publicly.

- 元件實作：`packages/react/src/components/`
- 設計交接 spec：`components/<group>/<name>/spec.md`
- Tokens：`packages/tokens/`
